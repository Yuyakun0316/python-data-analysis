# Python データ分析練習 (Pandas)

## 📂 学習内容

### 1. Pandas基礎 (`01_pandas_basics/`)
SQLの基本的な操作をPandasで実装し、対比学習を行いました。
- **データ抽出**: `WHERE` 句に相当するフィルタリング
- **集計**: `GROUP BY` に相当する `groupby` と `sum`
- **計算**: 列同士の演算による新しい列の追加

### 2. Pandas応用 - 独自ロジックの適用 (`02_advanced_pandas/`)
標準的な集計だけでなく、Pythonの関数を用いた柔軟なデータ加工を実装しました。
- **`apply` メソッド**: `if` 文などの条件分岐を含む独自関数をデータフレーム全体に適用
- **フラグ付け**: 価格に応じた「商品ランク（高級/低価格）」の自動付与

### 3. データクリーニング (`03_data_cleaning/`)
実務で必須となる「汚いデータ」を綺麗に整える（前処理）手法を実装しました。
- **重複削除 (`drop_duplicates`)**: 重複して記録された不要な行の削除
- **欠損値処理 (`fillna`, `dropna`)**: 空欄（NaN）を特定の文字（「不明」など）で埋める、または不要な行を削除する処理
- **フィルタリングとソート**: 特定条件によるデータの絞り込みと、金額順などの並び替えの実装

### 4. データの紐付け・結合 (`04_data_merging/`)
実務で頻出する「別々の表を一つに合体させる」処理を実装しました。VLOOKUP関数の上位互換として活用可能です。
- **内部/左結合 (`pd.merge`)**: 共通のキー（顧客IDなど）を軸に、売上データと顧客マスタを紐付け
- **列の整理 (`drop`)**: 結合後に不要となった列を削除し、レポート用に出力を最適化

### 5. 高度な集計レポート作成 (`05_data_aggregation/`)
ピボットテーブルのように、特定の項目でデータをまとめ、多角的な分析を行う手法を実装しました。
- **多機能集計 (`agg`)**: `sum`（合計）や `count`（件数）など、複数の集計処理を一度に実行
- **インデックスのリセット (`reset_index`)**: 集計後のデータを「平らな表」に戻し、Excel出力を容易にする処理

### 6. データの可視化 (`06_data_visualization/`)
外部CSVデータからビジネスの傾向を分析し、視覚的なレポートを自動生成する手法を実装しました。
- **CSV読み込み (`read_csv`)**: 外部ファイルからのデータインポート
- **グラフ描画 (`matplotlib`)**: 折れ線グラフによるトレンドの可視化
- **画像出力 (`savefig`)**: 分析結果を画像ファイル(PNG)として保存・自動出力

## 🛠 動作環境
- macOS
- Python 3.12.1
- Pandas
- Openpyxl (Excel出力用)
