# Python データ分析練習 (Pandas)

## 📂 学習内容

### 1. Pandas基礎 (`01_pandas_basics/`)
SQLの基本的な操作をPandasで実装し、対比学習を行いました。
- **データ抽出**: `WHERE` 句に相当するフィルタリング
- **集計**: `GROUP BY` に相当する `groupby` と `sum`
- **計算**: 列同士の演算による新しい列の追加

### 2. Pandas応用 - 独自ロジックの適用 (`02_advanced_pandas/`)
標準的な集計だけでなく、Pythonの関数を用いた柔軟なデータ加工を実装しました。
- **`apply` メソッド**: `if` 文などの条件分岐を含む独自関数を、データフレーム全体に適用
- **フラグ付け**: 価格に応じた「商品ランク（高級/低価格）」の自動付与

### 3. データクリーニング (`03_data_cleaning/`)
実務で必須となる「汚いデータ」を綺麗に整える（前処理）手法を実装しました。
- **重複削除 (`drop_duplicates`)**: 重複して記録された不要な行の削除
- **欠損値処理 (`fillna`, `dropna`)**: 空欄（NaN）を 0 で埋める、または特定の列に空欄がある行を削除する処理
- **データ型変換 (`astype`)**: 小数点を含む数値（float）を整数（int）に変換し、見た目とメモリ効率を最適化

## 🛠 動作環境
